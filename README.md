# ADS1256.multi
An Arduino-compatible library for the ADS1256 24-bit analog-to-digital converter. The idea of this fork is to create a library for managing multiple ADS1256 at the same time.

For usage, see the documentation: https://github.com/CuriousScientist0/ADS1256/blob/main/extras/ADS1256_ArduinoLibrary_Documentation_20251023.pdf
